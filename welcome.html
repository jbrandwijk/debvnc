Congratulations

If you are reading this, you sucessfully installed the docker container and logged into it. At your disposal is a very very basic window manager (blackbox), firefox and xterm (a terminal). You can install additional software using apt-get from xterm. Xterm is accessable through the context menu in blackbox (right click in an empty area on the "desktop").

This container is built upon Debian unstable. So the package manager is pointing to that repository as well. Please refer to apt's manual paged for fore information.
